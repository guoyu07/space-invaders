Creates a bullet with a starting position.

It takes the x-coordinate from its firing ship and an y-coordinate just above, under the ship for a playership, enemyship respectively.

Parameters
playertrue if player fired, false if enemy fired
poscurrent position of the firing ship
sizethe size of the bullet, default: w = 1, h = 1
speedthe speed with which the bullet moves each step: default = 2
