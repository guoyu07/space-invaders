Detailed Description

 Sets up a SFML Game and runs it.
Author
Olivier Brewaeys

Every level can be seen as one game and its corresponding world. This class also holds the game state, like the elapsed time and delegates user requests to the model which updates the view. It is the main controller function and runs the main/game loop.
